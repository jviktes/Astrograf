# Summary

Modular planetary gerabox  1024:1 (4:1, 16:1, 64:41, 256:1)  for NEMA17 stepper motors

Design in FreeCAD source on https://github.com/romanhujer/Modular-gearbox

How to this working? See my video: https://youtu.be/6pDPNkWU2vo

Other components is:

- 1x NEMA17 Stepper motor
- 15x 2 x 6mm [Self Tapping Screw](https://www.ebay.com/itm/301828057475)
- 4x M3 x 80mm Threaded rod 
- 3x M3 x 8mm Screw 
- 7x M3 Nuts
- 1x M5 Nuts
- 1x M5 x 20mm Screw
